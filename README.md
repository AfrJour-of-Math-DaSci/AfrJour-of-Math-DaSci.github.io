# AJMDS — African Journal of Mathematics and Data Science

Source for the journal's website, served by GitHub Pages at
<https://afrjour-of-math-dasci.github.io/>.

AJMDS is a forthcoming diamond open-access, peer-reviewed journal in the
mathematical and computational sciences, treating African problems and African
data as a primary focus. Launch planned for Q1 2027.

## Repository layout

- `index.html` — the landing page
- `ajmds-*.svg` — brand assets (seal, horizontal and vertical lockups, favicon)
- `.nojekyll` — disables Jekyll; files are served as-is
